Task Management API

  • Tech Stack:
    • Go
    • Bun
    • GraphQL
    • PostgreSQL
    • Git
  • Project: : Project Link

Developed a GraphQL-based API for the Task Tracker API using gqlgen. Go was utilized for the backend implementation, while Bun served as the ORM, and Postgresql was used for storage. Go migrations and Makefile were used for handling migrations and providing instructions, and Git was employed for tracking changes.
